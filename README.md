# Project I.G.I. SaveGame Mod

## Overview
This mod provides a mid-mission save/load system for *Project I.G.I. 1*, allowing players to save their progress at any point during the mission. This guide explains how to install the mod, provides an overview of the architecture, and outlines how to contribute to the project.

## Features
- **Save Anywhere**: Players can save their game at any point during missions, providing flexibility and reducing frustration.
- **Load Saves**: Easily load previously saved games without restrictions.
- **User-Friendly Interface**: The mod integrates smoothly with the game's existing UI for an enhanced user experience.
- **Robust Architecture**: Built on a solid framework to ensure stability and performance.

## Installation
1. **Download the Mod**: Visit the [releases page](https://github.com/Nage1993/Project-IGI-SaveGame-Mod/releases) to download the latest version of the mod.
2. **Extract Files**: Unzip the downloaded file to your preferred location.
3. **Copy Files**: Place the mod files into the main directory of your Project I.G.I installation (usually `C:\Program Files\Project IGI\`).
4. **Run the Game**: Launch Project I.G.I., and the mod should be active.

## Architecture Overview
The mod is designed with a focus on modular programming:
- **Core Module**: Handles the main save and load functionality.
- **UI Module**: Manages user interactions and displays save/load options in a non-intrusive manner.
- **Data Management**: Responsible for reading and writing save data to disk while ensuring data integrity.

This architecture promotes easy updates and maintenance while allowing contributors to easily add features or fix bugs.
